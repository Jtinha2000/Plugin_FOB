# Plugin FOB’s

> Esse repositório contém uma extensão desenvolvida para o jogo Unturned. A qual adiciona a funcionalidade de FOB 's ao jogo, semelhante ao jogo SQUAD. 

# O’que são FOB’s?

> A sigla FOB’s em portugues remete a “Base Operacional Avançada”, sendo no jogo uma barricada qualquer de ID Y, na qual jogadores do mesmo time podem, se não houver inimigos próximos, se teletransportar. Ideal para servidores de guerra.
